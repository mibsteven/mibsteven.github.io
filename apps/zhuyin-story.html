注音故事

家裡的故事又看完了嗎？注音故事運用 AI 生成故事，持續為孩子提供適合閱讀的材料。

特色

  • 運用 AI 生成故事內容，提供適合孩子閱讀的材料
  • 結合感官輪、108 課綱議題與成語運用
  • 搭配插畫，讓孩子在愉快體驗中提升閱讀能力
  • 自研閱讀器可為生成內容加上注音，並調整成適合孩子閱讀的間距與行高
  • 內建朗讀功能，可作為故事時間補充

AI 使用方式

在支援 On-Device AI 的機型上，可以離線使用主要生成能力；若希望取得更豐富的內容，也可使用外接語言模型選項。

注音故事

注音故事 is an AI-assisted reading app that creates Zhuyin-supported stories for children when families need fresh reading material.

Features

  • Generates story content suitable for children
  • Combines sensory prompts, Taiwan curriculum topics, and idiom practice
  • Pairs stories with illustrations for a friendlier reading experience
  • Uses a custom reader to add Zhuyin and tune spacing for young readers
  • Includes narration features for story time support

AI Usage

On supported devices, key generation features can run with on-device AI. The app also offers optional external language model support for richer content.
